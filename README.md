## La Cafeteria
Pagina cascaron de una cafetería en esta se muestra secciones como reservaciones, galería, menú, proceso del café y un acerca de, este proyecto fue creado con html y sass, además de utilizar gulp para el compilado de sass y optimización de imágenes.
